# PatroAsteroids Page

Este repositório contém o front-end da página **PatroAsteroids**, criada para exibir informações e estatísticas de jogadores do jogo PatroAsteroids, que se comunica com o back-end via API.

## Funcionalidades

- Exibe uma tabela com dados de todos os jogadores registrados, como nome, maior pontuação, moedas coletadas, etc.
- Comunicação em tempo real com o servidor para recuperar e exibir estatísticas de jogadores.
- Interface simples e responsiva, facilitando a visualização das informações em diferentes dispositivos.

## Tecnologias Utilizadas

- **HTML5**: Estruturação da página.
- **CSS3**: Estilização da página com um layout limpo e responsivo.
- **JavaScript (Fetch API)**: Comunicação com o back-end para obtenção de dados via API.
- **Express.js**: Back-end utilizado para gerenciar e fornecer os dados dos jogadores.

## Como Executar

1. Clone o repositório:

   ```bash
   git clone https://github.com/SEU_USUARIO/PatroAsteroidsPage.git
   ```

2. Abra o arquivo `index.html` no navegador.

3. Certifique-se de que o servidor Express.js está rodando para a comunicação adequada com a página.

## Estrutura do Repositório

- `index.html`: Página principal que exibe as estatísticas dos jogadores.
- `styles.css`: Arquivo de estilização para a interface.
- `app.js`: Arquivo JavaScript responsável pela interação com a API do back-end.

## Back-end
